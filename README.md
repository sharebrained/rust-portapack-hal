# portapack-hal

Hardware abstraction layer (HAL) for the [PortaPack] add-on for the [HackRF One] software-defined radio.

[PortaPack]: http://www.sharebrained.com/portapack/
[HackRF One]: http://www.greatscottgadgets.com/hackrf/
